# Configurações:

- Adicionar endereço da página
```bash
/application/config/config.php
$config['base_url'] = '';
```

- Adicionar nome do banco, usuário e senha
```bash
/application/config/database.php
'username' => '',
'password' => '',
'database' => '',
```

- Importar sql.sql para seu DB
```bash
sql.sql
```

- Endereço, e-mail e senha para login
```bash
SEU-ENDEREÇO/login
mail@mail.com
Euro2022
```
